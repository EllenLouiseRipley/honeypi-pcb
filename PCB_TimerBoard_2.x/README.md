# HoneyPI-PCB TimerBoard 2,x

Extension board for the HoneyPi-PCB 2.x.  
Pitch lenght 2.54 mm for pins or...  
Designed to be mounted on a Raspberry Pi Zero

Feature list and bill of material is still on working.
